# AI-Data-Analysis-Agent
An AI-powered Data Analysis Agent built using Python, LangChain, Google Gemini, and Pandas. It enables users to analyze CSV and Excel datasets using natural language queries without writing SQL or Python code.
# AI Data Analysis Agent

## Overview

AI Data Analysis Agent is an intelligent assistant that allows users to analyze CSV and Excel datasets using natural language.

Instead of writing Python code or SQL queries, users can ask questions like:

- What is the total revenue?
- Show the top 5 sales days.
- Which region has the highest sales?
- What is the average order value?

The application automatically generates Python code using an LLM and performs data analysis with Pandas.

---

## Features

- Natural language querying
- CSV and Excel support
- Automatic data analysis
- Google Gemini integration
- LangChain Pandas Agent
- Sample dataset generation
- Interactive command-line interface
